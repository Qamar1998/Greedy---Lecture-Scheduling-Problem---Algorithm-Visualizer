Description
You have to enter a number of lectures with their start and finish times. The goal is to find the maximum number of mutually compatible lectures.
Solution
Choosing the lecture which is going to finish first. Then check if it compatible and not overlap with the prev selected lecture
Time Complexity : ⊝ ( n log n )
